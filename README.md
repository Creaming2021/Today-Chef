## :sparkles: 기대효과
- **건강하고 간편한 집밥 요리 횟수 증가**
- 유저들 간의 다양한 요리 **지식 공유**
- 스트리밍 서비스를 통한 **실시간 질의응답**
- 스트리밍 **음성 제어 서비스**로 요리를 진행하면서도 서비스를 편하게 사용 가능
- 스트리밍 STT기능과 AR 영상 자막 서비스 제공으로 **청각장애인도 편리하게 서비스 이용 가능**
- 향후 카테고리 확장을 통해 **다양한 분야에서 지식 공유 가능**
